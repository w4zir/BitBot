# Delivery Period Policy
**Objective:** Explain expected and actual delivery windows using shipment records.

### Delivery Period Lookup
* Accept either order ID (`ORD-*`) or tracking ID (`TRK-*`).
* Provide promised delivery time when available.
* Provide actual delivery time if delivered.

### Delay Communication
* If no shipment is found, ask customer to verify order/tracking reference.
* If delayed, include recorded delay reason when present.
